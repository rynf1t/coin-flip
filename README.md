# Coin Flip

Randomly generates a coin flip result with retro arcade-style animation and visual feedback showing heads or tails.

## Live Demo

[Try it here](https://coin-flip.itsryan.me/)

## Running Locally

```bash
git clone https://github.com/rynf1t/coin-flip.git
cd coin-flip
open index.html
```
